## Технические детали

### Используемые структуры данных
- **dataclass** для хранения информации о культуре
- **list** для хранения списка культур
- **dict** (опционально) для более сложной структуры, если понадобится

### Циклы в проекте
1. **Цикл ввода данных** - пользователь может добавлять несколько культур
2. **Цикл расчета** - суммирование урожая всех культур в списке
3. **Цикл отображения** - вывод списка культур в интерфейсе

### Формула расчета
```
Общий объем урожая = Площадь посева × Урожайность
Общий урожай за сезон = Сумма всех культур
```
