# Учет урожая

Приложение для ведения учета урожая различных культур на ферме. Позволяет фермеру вводить данные по нескольким культурам, хранить информацию о площади посева и урожайности, а также рассчитывать общий объем урожая за сезон.

## Описание

Программа использует структуру данных (dataclass) для хранения информации о каждой культуре:
- Название культуры
- Площадь посева (в гектарах)
- Урожайность (в тоннах на гектар)
- Общий объем урожая (рассчитывается автоматически)

Приложение предоставляет графический интерфейс на tkinter для удобного ввода данных и просмотра результатов.

## Требования

- Python 3.7 или выше
- tkinter (встроен в стандартную библиотеку Python)

## Установка

1. Клонируйте репозиторий или скачайте проект

```bash
venv\Scripts\activate  # Для Windows
```

## Запуск

Запустите приложение командой:

```bash
python main.py
```

## Использование

### Добавление культуры

1. Введите название культуры в поле "Название культуры"
2. Введите площадь посева в гектарах в поле "Площадь посева (га)"
3. Введите урожайность в тоннах на гектар в поле "Урожайность (т/га)"
4. Нажмите кнопку "Добавить культуру"

Культура будет добавлена в список и отобразится в области "Список культур".

### Расчет общего урожая

После добавления одной или нескольких культур:
1. Нажмите кнопку "Рассчитать общий урожай"
2. Общий объем урожая за сезон отобразится в нижней части окна и в диалоговом окне

### Очистка списка

Для удаления всех культур из списка:
1. Нажмите кнопку "Очистить список"
2. Подтвердите действие в диалоговом окне

## Примеры использования

### Пример 1: Добавление одной культуры

```
Название культуры: Пшеница
Площадь посева: 10.5
Урожайность: 3.2

Результат: Пшеница: 10.5 га × 3.2 т/га = 33.60 т
```

### Пример 2: Добавление нескольких культур

```
Культура 1:
- Название: Пшеница
- Площадь: 10.0 га
- Урожайность: 3.5 т/га
- Урожай: 35.00 т

Культура 2:
- Название: Ячмень
- Площадь: 5.0 га
- Урожайность: 2.8 т/га
- Урожай: 14.00 т

Культура 3:
- Название: Овес
- Площадь: 8.0 га
- Урожайность: 2.5 т/га
- Урожай: 20.00 т

Общий урожай за сезон: 69.00 т
```

## Особенности

- **Валидация данных**: Проверка корректности введенных данных (положительные числа, непустые названия)
- **Обработка ошибок**: Информативные сообщения об ошибках при некорректном вводе
- **Циклы**: Использование циклов для обработки списка культур и расчета общего урожая
- **Структуры данных**: Использование dataclass для хранения информации о культурах

## Формула расчета

```
Общий объем урожая культуры = Площадь посева × Урожайность
Общий урожай за сезон = Сумма урожая всех культур
```
